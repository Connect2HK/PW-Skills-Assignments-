Ordered List of HTML Tags

  1. <html> - The html tag is the root element of an HTML page. It contains all other elements in the document.
  2. <head> - The head tag contains meta-information about the HTML document, such as the title, character set, and links to stylesheets.
  3. <title> - The title tag defines the title of the HTML document, displayed in the browser's title bar or tab.
  4. <body> - The body tag contains the content of the HTML document, such as text, images, links, and other elements.
  5. <h1> to <h6> - The h1 to h6 tags are used to define headings, with h1 being the highest level and h6 the lowest.
  6. <p> - The p tag is used to define paragraphs of text.
  7. <a> - The a tag defines a hyperlink, which can link to another web page, file, or location within the same page.
  8. <img> - The img tag is used to embed images into an HTML document.
  9. <ul> - The ul tag defines an unordered list, which is typically displayed with bullet points.
  10. <ol> - The ol tag defines an ordered list, which is typically displayed with numbers or letters.
  11. <li> - The li tag is used to define a list item within an unordered (<ul>) or ordered list (<ol>).
  12. <div> - The div tag defines a division or section within an HTML document, often used as a container for styling or layout purposes.
  13. <span> - The span tag is an inline container used to mark up a part of a text or document for styling.
  14. <form> - The form tag is used to create an HTML form for user input, containing form elements like input fields, checkboxes, and buttons.
  15. <input> - The input tag is used to create interactive controls in an HTML form, such as text boxes, radio buttons, or checkboxes.
  16. <button> - The button tag defines a clickable button within an HTML form or document.
  17. <table> - The table tag is used to create a table in an HTML document, consisting of rows and columns.
  18. <tr> - The tr tag defines a row within an HTML table.
  19. <th> - The th tag defines a header cell within a table row, usually displayed as bold and centered.
  20. <td> - The td tag defines a standard data cell within a table row.
  21. <strong> - The strong tag is used to emphasize text with strong importance, usually rendered in bold.
  22. <em> - The em tag is used to emphasize text, typically rendered in italics.
